# Imersão_Filosofia

Imersão_Filosofia é um catálogo interativo feito com HTML, CSS e JavaScript, permitindo pesquisar filósofos, conceitos, obras e escolas por meio de busca e filtros. Inspirado na Imersão Dev, utiliza arrays, objetos e manipulação do DOM para exibir cards dinâmicos de forma simples, útil e criativa.

## Tecnologias

- HTML5  
- CSS3  
- JavaScript (ES6+)

## Como executar

1. Clone o repositório ou faça o download dos arquivos.
2. Abra o arquivo `index.html` no navegador.
3. Use a busca e os filtros para explorar os conteúdos de filosofia.

## Critérios da Imersão atendidos

- **Utilidade**: apoio ao estudo de filosofia para estudantes e professores.  
- **Criatividade**: aplicação do modelo de catálogo da Imersão em um tema filosófico.  
- **Eficácia**: filtros e busca funcionam com arrays, objetos e DOM.  
- **Apresentação**: layout limpo, responsivo e organizado em cards.
